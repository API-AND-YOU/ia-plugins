# Développement

## Claude Code

```
/plugin marketplace add ./
/plugin install devexpress-documentation@api-and-you-plugins
```

## GitHub Copilot CLI

```
/plugin marketplace add ./
/plugin install devexpress-documentation@api-and-you-plugins
```

> **Note :** Le CLI Copilot n'enregistre pas automatiquement les serveurs MCP déclarés dans `plugin.json` lors de l'installation du plugin. Après installation, configurez manuellement le serveur MCP DevExpress avec la commande :
>
> ```
> /mcp add dxdocs https://api.devexpress.com/mcp/docs
> ```
